# Deno Space Traders

A command line app for interacting with [spacetraders](spacetraders.io)

## Usage

- Install [deno](https://deno.land/)
- `deno run --unstable --allow-net index.ts`

## Currently supports:

- Creating a new user
- Finding a loan
- Taking out a loan
